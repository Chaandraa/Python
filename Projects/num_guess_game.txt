# Number Guessing Game Project Report (Python)

## Project Title

Number Guessing Game using Python

---

# 1. Project Objective

The objective of this project is to create an interactive guessing game where the computer generates a random number and the user tries to predict it in minimum attempts.

The project focuses on improving:

* Logical thinking
* Input validation
* Loop control
* Decision making
* User interaction

---

# 2. Problem Statement

Traditional beginner programs are static and do not simulate real user interaction.

This project solves that by:

* Generating unpredictable values
* Accepting user guesses
* Providing feedback
* Counting attempts

The user continues guessing until the correct answer is found.

---

# 3. Features Implemented

✔ User chooses difficulty range

✔ Random number generation

✔ Input validation

✔ Infinite game loop

✔ Guess counter

✔ Higher/lower hints

✔ Final result display
