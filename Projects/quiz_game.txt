# Quiz Game Project Report (Python)

## Project Title

Terminal-Based Quiz Game using Python

---

# 1. Project Objective

The objective of this project is to create an interactive quiz application where users answer multiple questions and receive a score and percentage at the end.

This project demonstrates the application of core Python concepts including decision making, user input handling, variables, string manipulation, and program flow control.

---

# 2. Problem Statement

Traditional learning methods can become repetitive and less engaging.

This project solves that by creating a command-line quiz system that:

* Interacts with users
* Evaluates responses
* Tracks scores
* Displays performance

---

# 3. Features Implemented

✔ Welcome screen

✔ User start confirmation

✔ Multiple quiz questions

✔ Answer validation

✔ Score tracking
