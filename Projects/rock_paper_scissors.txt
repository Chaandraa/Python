# Rock Paper Scissors Game – Project Report

## Project Title

Rock Paper Scissors Game using Python

---

# 1. Project Objective

The objective of this project is to create an interactive game where a user competes against the computer.

The system:

* Accepts player choices
* Randomly generates computer choices
* Compares outcomes
* Tracks wins
* Displays results

This project strengthens logical thinking and interactive programming.

---

# 2. Problem Statement

Traditional programs execute once and terminate.

This project creates a continuously running interactive system that:

* Handles repeated user actions
* Simulates computer decisions
* Maintains scores

---

# 3. Features Implemented

✔ User input

✔ Computer random selection

✔ Input validation

✔ Score tracking

✔ Winner determination

✔ Replay through loop

✔ Quit option
